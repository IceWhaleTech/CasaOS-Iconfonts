# CasaOS Icon fonts library

current version: 1.0.4

## Install

```bash
npm install iconfonts-casaos
```

or

```bash
pnpm install iconfonts-casaos
```

or

```bash
yarn add iconfonts-casaos
```

## How to use

```css
@import '~iconfonts-casaos/casaos-iconfonts.css';
```
